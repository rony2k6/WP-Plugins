Generate Embed Code to share image on any site.

== Description ==

* This plugin generate embed code at front-end.
* This plugin gives shortcode to place embedcode generator at front-end.

== Installation ==

1. Upload 'hope_et' folder to the '/wp-content/plugins/' directory or upload 'hope_et.zip' through WordPress admin panel
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcde "[hope_embed]" to any post/page content or text widget* (*If text widget has shortcode support).
